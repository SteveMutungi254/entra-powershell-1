# New-EntraApplicationPassword

## Synopsis

Adds a strong password to an application.

## Syntax

```powershell
New-EntraApplicationPassword 
 -ObjectId <String> 
 -PasswordCredential <PasswordCredential> 
 [<CommonParameters>]
```

## Description

Adds a strong password to an application.

## Examples

### Example 1: Add a password to an application

```powershell
Connect-Entra -Scopes 'Application.ReadWrite.All' #Delegated Permission
Connect-Entra -Scopes 'Application.ReadWrite.OwnedBy' #Application Permission
$params = @{
    ObjectId = '00001111-aaaa-2222-bbbb-3333cccc4444'
    PasswordCredential = @{
        displayname = 'mypassword'
    }
}

New-EntraApplicationPassword @params
```

```Output
CustomKeyIdentifier DisplayName EndDateTime           Hint KeyId                                SecretText                       StartDateTime
------------------- ----------- -----------           ---- -----                                ----------                       -------------
                    mypassword  10/28/2021 3:57:37 PM EQ:  bbbbbbbb-1c1c-2d2d-3e3e-444444444444 <my-secret-text> 10/28/2019 3:57:37 PM
```

This command adds a password to the specified application.

The secretText property in the response object contains the strong passwords/secrets generated by Microsoft Entra ID that are 16-64 characters in length. 

There is no way to retrieve this password in the future.
