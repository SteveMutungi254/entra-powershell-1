# Get-EntraBetaDirectorySetting

## Synopsis
Gets a directory setting.

## Syntax

### GetQuery (Default)

```powershell
Get-EntraBetaDirectorySetting
 [-All]
 [-Top <Int32>]
 [-Property <String[]>]
 [<CommonParameters>]
```

### GetById

```powershell
Get-EntraBetaDirectorySetting
 -Id <String>
 [-All]
 [-Property <String[]>]
 [<CommonParameters>]
```

## Description
The Get-EntraBetaDirectorySetting cmdlet gets a directory setting from Azure Active Directory (AD).
